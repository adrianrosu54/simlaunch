# SimLaunch

## FTC Decode Flywheel Launcher simulation

Web visualiser application for robotic launcher physics in the
FTC Decode Season made by team #19075 Clockworks.

It features live interactive charts for parameters of the launcher,
including a trajectory plot, field view display and impact error
calculation.

The calculations take into account gravity and basic drag force,
along with the flywheel launch efficiency.

Project made with Astro and React.
