# Flywheel Launcher Simulation Dashboard

Web Dashboard application that visualises the physical model of the flywheel
launcher. It computes and logs the simulated trajectory of the artifact,
providing an interactive feedback for the projected flight and estimated
impact location.

It features the side trajectory display, with 3 dynamic graphs for the
trajectory, velocity and acceleration.

The control panel features over 2 custom controls for adjusting the
flywheel velocity and launcher heading angle
